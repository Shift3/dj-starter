# Folder Structure

After creation, your project should look something like this:

```bash
my_project/
├── my_project/
│   ├── agents/
│   ├── asgi.py
│   ├── core/
│   ├── notification_system/
│   ├── settings/
│   ├── urls.py
│   ├── users/
│   └── wsgi.py
├── LICENSE
├── README.md
├── requirements/
│   ├── base.txt
│   ├── dev.txt
│   ├── prod.txt
│   └── test.txt
├── scripts/
    └── update-circleci.sh
```
